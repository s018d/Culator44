# سجل (Sajil) - تطبيق الآلة الحاسبة السرية

تطبيق أندرويد احترافي متنكر في شكل آلة حاسبة، يتيح تسجيل الفيديو والصوت في الخلفية بشكل سري.

## المميزات
- واجهة آلة حاسبة حقيقية تعمل بالكامل.
- نظام دخول سري بواسطة كود خاص (`79560`).
- تسجيل فيديو وصوت في الخلفية (Foreground Service).
- التوقف التلقائي عند انخفاض البطارية.
- إدارة كاملة للتسجيلات (تشغيل، مشاركة، حذف).
- تقنيات حديثة: Jetpack Compose, CameraX, Room DB, MVVM.

## المتطلبات التقنية
- Android 10 (API 29) to Android 15 (API 35).
- Kotlin 2.0.21.
- Gradle 8.7.3.
